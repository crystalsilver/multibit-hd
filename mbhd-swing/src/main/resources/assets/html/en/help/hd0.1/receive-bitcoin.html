Request payment

To request digibyte from someone else they need a "receiving address". Multibit-HD gives you a new receiving address each time you select "Send/Request" and then "Request".

  1. On the "Request payment" screen you don't have to fill anything in. You can just copy the the address, by:
    • highlighting the address and using CTRL-C, or
    • selecting the copy icon, or
    • selecting the QR code icon, to get a QR code that a Bitcoin wallet on a mobile phone could use to send you digibyte.
  2. However, you can add more information about the digibyte you want to receive. For instance, you could
    • enter a private note (e.g. "Payment for software development") to help you keep track of why you're receiving digibyte,
    • specify the amount you expect (either in digibyte or in the local currency),
    • add a label for the QR code. The QR code contains the address, the label and the amount all in a convenient format.
  3. Click "Finish" to go back to the Send/Request screen.
