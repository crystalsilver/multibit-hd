Raise an issue

Please only raise an issue if you have genuinely searched hard for an answer to your problem and failed to come up with an answer

We welcome bug reports and need your feedback to improve the software for everyone else.

Use the approach below to get the fastest response from the support team:

  1. Copy the following text: 
      Bug report
  Operating System:
  DigiBit HD Version:
  Steps to replicate the problem:
  1. Provide steps to reproduce
  2. Provide a screen shot (use CMD+SHIFT+4+SPACE+Click on Mac, ALT+PrtScr on Windows then paste)
  3. Avoid personally identifying information
  2. Re-visit the DigiBit HD Issue reporting center
  3. Click on the "New Issue" button.
  4. Paste the template text and fill it in for your computer. The more detail you can give the quicker we can address the problem. Screen shots are often very useful but be sure to hide any sensitive information since they will be public.
